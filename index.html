    m0sk1t::engine

    Hello there! This is a yet another flexible library for HTML5 <canvas> tag manipilation (which also containing framework for fast development of 2d games) named m0sk1t::engine.

    Description:

    m0sk1t::engine consist of one base class named core and several helper classes, such as utils, primitive, assets, physics, input. Full description of classes is situated in reference folder:
